# Pokemon API

## Functionality
- Returns Pokemon information based on <a src="https://pokeapi.co/">PokeApi</a> data.
- Create Pokemon teams

## Installation commands
- cp .env.example .env
- composer install
- npm install
- php artisan migrate
- php artisan import:pokemon --gen 1
- php artisan test

### Import command variables
- --gen 1
- --name pikachu
- --id 151
